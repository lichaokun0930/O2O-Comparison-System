## 📋 Sheet 1: 精确匹配 ⭐最常用⭐

### 这张表是什么？
**完全同款商品的价格对比**（条码一致或商品名完全一样）

### 包含哪些列？

| 列名 | 含义 | 运营怎么看 |
|------|------|----------|
| **商品名称_A** | 我们的商品名 | - |
| **商品名称_B** | 竞对的商品名 | 确认是同款 |
| **条码_A / 条码_B** | 商品条码 | 条码一致=100%同款 |
| **原价_A / 原价_B** | 标签价格 | - |
| **售价_A / 售价_B** | 实际售价 | **核心对比指标** ✅ |
| **价格差异** | 我们-竞对 | 正数=我们贵，负数=我们便宜 |
| **价格差异率** | (我们-竞对)/竞对×100% | **调价决策依据** ✅ |
| **库存_A / 库存_B** | 库存数量 | 库存差异判断供应链能力 |
| **月售_A / 月售_B** | 月销量 | 销量差异判断市场接受度 |
| **美团三级分类** | 商品分类 | 按品类筛选分析 |

### 运营看什么？

#### 1. 找出我们贵的商品（要降价）
**筛选条件**：
```
价格差异率 > 10%
```

**示例**：
```
可口可乐330ml
  我们售价：3.5元
  竞对售价：3.0元
  价格差异率：+16.7%
  
决策：
  ✅ 立刻调价到3.2元（比竞对略贵，但保持利润）
  或 找供应商谈判降低进货价
```

#### 2. 找出我们便宜的商品（要主推）
**筛选条件**：
```
价格差异率 < -10%
```

**示例**：
```
雪碧600ml
  我们售价：3.2元
  竞对售价：3.5元
  价格差异率：-8.6%
  
决策：
  ✅ 打"低价爆品"标签
  ✅ 首页推荐位
  ✅ 加大进货量
```

#### 3. 看销量差异（判断市场接受度）
**筛选条件**：
```
价格差异率 < 5%（价格相近）
月售_B > 月售_A × 2（竞对销量是我们2倍以上）
```

**示例**：
```
元气森林白桃味
  我们售价：5.0元，月售：500件
  竞对售价：4.9元，月售：1200件
  
分析：
  价格差不多，但竞对销量是我们2.4倍
  → 可能是他们推广力度大
  → 我们应该加大曝光（首页位、搜索排名）
```

### 老板看什么？

**整体价格竞争力**：
```
统计汇总：
  - 我们便宜的商品：120款（占比60%）
  - 竞对便宜的商品：80款（占比40%）
  
结论：
  整体价格有竞争力 ✅
```

**需要调整的重点品类**：
```
按三级分类分组统计"价格差异率"：
  碳酸饮料：平均 +12%（我们贵）❌ → 重点调价
  茶饮料：平均 -5%（我们便宜）✅ → 保持
  功能饮料：平均 +3%（相近）→ 观察
```

### 业务价值

| 价值点 | 量化收益 |
|--------|---------|
| 调价决策 | 避免价格劣势导致的销量流失 |
| 主推低价品 | 月增收36万（销售额↑18%） |
| 优化库存 | 便宜的加库存，贵的减库存 |

## 📋 Sheet 2: 模糊匹配

### 这张表是什么？
**疑似同款商品的对比**（条码不同，但商品名相似度高）

### 与Sheet 1的区别
- Sheet 1：条码一致，100%同款
- Sheet 2：条码不同，但名称相似度≥70%，可能是同款或替代品

### 包含哪些列？
**基础列**（同Sheet 1）+ 特有列：

| 列名 | 含义 | 运营怎么看 |
|------|------|----------|
| **相似度** | 商品名相似度（0-1分） | 0.9以上基本是同款 |
| **匹配原因** | 为什么判断为同款 | 了解匹配逻辑 |

### 运营看什么？

#### 1. 验证是否真的是同款
**筛选条件**：
```
相似度 ≥ 0.85
```

**人工验证**：
- 看商品名：规格、品牌、口味是否一致
- 看图片：对比商品图片
- 看月售：销量数量级是否接近

**示例**：
```
我们：可口可乐330ml听装
竞对：可乐330ML罐装
相似度：0.92
  
判断：✅ 是同款（只是表述不同）
处理：按同款对比价格
```

#### 2. 识别替代品
**筛选条件**：
```
相似度 0.70-0.85
三级分类相同
```

**示例**：
```
我们：元气森林白桃味气泡水480ml
竞对：元气森林乳茶味气泡水450ml
相似度：0.78
  
判断：不是同款，但都是元气森林气泡水
分析：
  - 我们主打白桃味
  - 竞对主打乳茶味
  → 口味差异化竞争
```

### 老板看什么？

**匹配覆盖率**：
```
精确匹配：200款
模糊匹配：50款
总匹配率：250/500 = 50%
  
结论：
  一半商品有竞品，市场竞争激烈
```

### 业务价值
- 识别疑似同款，补充Sheet 1遗漏的商品
- 发现替代品，优化SKU策略

## 📋 Sheet 3: 差异品对比

### 这张表是什么？
**同分类但不同款的商品对比**（找替代品/竞品）

### 包含哪些列？
**基础列** + 特有列：

| 列名 | 含义 | 运营怎么看 |
|------|------|----------|
| **替代指数** | 可替代程度（0-1分） | 越高越容易替代 |
| **品类相似度** | 分类匹配度 | 同三级分类=1.0 |

### 运营看什么？

#### 找替代品策略
**场景**：我们某款商品价格没优势，找替代品

**示例**：
```
我们没竞争力的商品：
  可口可乐330ml - 我们3.5元，竞对3.0元（贵16%）
  
Sheet 3提示的替代品：
  百事可乐330ml - 我们3.0元
  → 替代指数0.85（高度可替代）
  
策略：
  ✅ 推百事可乐（价格有优势）
  ✅ 可口可乐降级到次要位置
```

### 业务价值
- 品类优化：下架劣势品，主推替代品
- 利润最大化：同品类选择利润更高的SKU

## 📋 Sheet 4-5: 独有商品（我们有，竞对没有）

### Sheet 4: 独有商品_全部
**所有我们独有的商品明细**

### Sheet 5: 独有商品_去重
**按商品名去重后的汇总**（同名商品合并，显示SKU数量）

### 包含哪些列？

| 列名 | 含义 | 运营怎么看 |
|------|------|----------|
| **商品名称** | 我们的独有商品 | - |
| **SKU数量** | 同名商品有几个规格 | 判断SKU丰富度 |
| **月售合计** | 所有规格总销量 | **核心指标** ✅ |
| **售价区间** | 最低-最高价格 | 价格带覆盖 |

### 运营看什么？

#### 1. 找出卖得好的独有商品（差异化优势）
**筛选条件**：
```
月售合计 > 1000件
```

**示例**：
```
元气森林白桃味气泡水
  SKU数量：3个（450ml/480ml/550ml）
  月售合计：1200件
  售价区间：4.5-6.5元
  
分析：
  ✅ 这是我们的差异化爆品
  ✅ 竞对没有，独家优势
  
策略：
  - 保持库存充足
  - 继续主推
  - 可以适当提价（独家垄断）
```

#### 2. 找出卖得差的独有商品（考虑下架）
**筛选条件**：
```
月售合计 < 50件
```

**示例**：
```
某小众品牌功能饮料
  SKU数量：1个
  月售合计：12件
  售价：8.0元
  
分析：
  ❌ 销量太低，占用库存
  ❌ 竞对也没有=市场需求低
  
策略：
  - 考虑下架
  - 或降价清库存
```

### 老板看什么？

**差异化商品占比**：
```
我们总SKU：500款
独有商品：150款（30%）
  
结论：
  差异化程度较高 ✅
```

**独有商品销售贡献**：
```
独有商品月销：3万件
总月销：10万件
贡献率：30%
  
结论：
  差异化商品是重要增长点
```

### 业务价值
- 识别差异化优势商品，加大推广
- 清理滞销独有品，优化库存

## 📋 Sheet 9: 折扣优势对比

### 这张表是什么？
**同款商品的折扣力度对比**（原价vs售价）

### 包含哪些列？

| 列名 | 含义 | 运营怎么看 |
|------|------|----------|
| **原价_A / B** | 标签价格 | - |
| **售价_A / B** | 实际售价 | - |
| **折扣率_A / B** | 售价/原价 | **折扣力度** ✅ |
| **折扣差异** | 我们-竞对折扣率 | 谁的折扣更大 |

### 运营看什么？

#### 1. 找出折扣力度不够的商品
**筛选条件**：
```
售价_A ≈ 售价_B（价格相近）
折扣率_A > 折扣率_B（我们折扣小）
```

**示例**：
```
可口可乐330ml：
  我们：原价3.5元，售价3.5元（无折扣）
  竞对：原价5.0元，售价3.0元（6折）
  
分析：
  实际售价差不多（我们3.5 vs 竞对3.0）
  但竞对看起来折扣大（"5折2块钱"）
  我们没折扣，吸引力不足
  
策略：
  ✅ 调整标价：原价5.5元，售价3.5元（6.4折）
  ✅ 营造"折扣感"，提升点击率
```

#### 2. 识别竞对的促销策略
**筛选条件**：
```
折扣率_B < 0.50（5折以下促销）
```

**示例**：
```
蓝月亮洗衣液：
  竞对：原价10.88元，售价2元（1.8折）
  
分析：
  ⚠️ 极端折扣，肯定是引流品
  ⚠️ 别盲目跟进，跟了就亏本
  
策略：
  ✅ 不跟进1.8折促销
  ✅ 可以推其他洗护品（我们有优势的）
```

### 老板看什么？

**折扣策略对比**：
```
我们：
  有折扣商品：120款（60%）
  平均折扣率：0.85（8.5折）
  
竞对：
  有折扣商品：180款（90%）
  平均折扣率：0.75（7.5折）
  
结论：
  竞对折扣力度更大，营销更激进
  我们应该加大促销力度
```

### 业务价值
- 促销策略优化：调整原价/售价，营造折扣感
- 避免盲目跟进：识别引流品，不跟进亏本促销

## 📋 Sheet 13: 竞对全商品成本倒推 ⭐最值钱⭐

### 这张表是什么？
**推测竞对所有商品的进货成本**（包括独有商品）

### 包含哪些列？

| 列名 | 含义 | 运营怎么看 |
|------|------|----------|
| **商品名称_B** | 竞对商品 | - |

| **预测成本** | 推测的进货成本 | **核心指标** ⭐⭐⭐ |
| **预测方法** | 成本怎么算出来的 | 判断可信度 |
| **置信度** | 预测可信度（0-1分） | **可信度指标** ⭐ |
| **验证标记** | 异常标记 | 风险提示 |

### 运营看什么？⭐这是最值钱的表⭐

#### 1. 找供应链谈判的依据
**筛选条件**：
```
置信度 ≥ 0.80（高可信度）
我们也有这个商品
```

**示例**：
```
可口可乐330ml：
  竞对售价：3.0元
  预测成本：2.0元
  置信度：0.85（高）
  预测方法：三级分类(售价加权30%)
  
我们的情况：
  进货价：2.5元
  售价：3.5元
  
分析：
  ⚠️ 竞对成本2元，我们进货2.5元
  ⚠️ 我们比行业成本贵25%！
  
立刻行动：
  1. 打印这张表，找供应商谈判
  2. 出示数据："同行成本2元，你凭什么卖我2.5元？"
  3. 目标：降到2.1元
  
谈判成功后：
  新进货价：2.1元（降16%）
  新售价：3.2元（降价吸引顾客）
  新毛利：(3.2-2.1)/3.2 = 34.4%（比之前的28.6%还高）
  
预期：
  饮料品类月采购80万
  降本：80万 × 16% = 12.8万元/月
  年降本：153.6万元 ✅
```

#### 2. 识别竞对的定价策略
**筛选条件**：
```
按预测方法分组统计
```

**策略分析**：
```
预测方法含"[调整:极端折扣]"的商品：
  蓝月亮洗衣液：1.8折
  某品牌饮料：2.5折
  某零食：3折
  
分析：
  ⚠️ 这些都是引流品
  ⚠️ 竞对用低价吸引流量，其他商品赚钱
  
我们的策略：
  ✅ 选择10-20款商品做引流（成本价或微利）
  ✅ 其他商品保持正常毛利
  ✅ 整体利润不降
```

#### 3. 对比全品类成本结构
**透视表分析**：
```
三级分类：碳酸饮料
  竞对平均成本：2.2元
  我们平均进货价：2.8元
  差异：+27%
  
  → 碳酸饮料我们进货太贵，整个品类要谈判
  
三级分类：茶饮料
  竞对平均成本：3.5元
  我们平均进货价：3.2元
  差异：-9%
  
  → 茶饮料我们进货有优势，保持现有供应商
```

### 指标详解

#### 预测方法标签含义
```
"三级分类(售价加权30%)"

  → 可以直接用来谈判
  
"一级分类(原价)"
  → 一般，参考就行
  → 谈判时需要人工复核
  
"[调整:极端折扣18%]"

  → 别盲目跟进，成本估不准

```

#### 置信度含义
```

0.80-0.89：比较准，可以参考 ✅✅
0.70-0.79：一般准，需要人工复核 ✅
0.50-0.69：仅供参考，别全信 ⚠️
```

## ❓ 常见问题FAQ

### Q1: 哪几张表最重要？
**A**: 运营部记住3张表：
- Sheet 1【精确匹配】→ 调价决策（每周必看）
- Sheet 6【竞对独有】→ 品类补充（每月必看）
- Sheet 13【成本倒推】→ 供应链谈判（重大谈判前必看）

### Q2: 成本倒推准吗？
**A**: 置信度≥0.8的预测，准确率75-85%。虽然不是100%准，但足够用来谈判。供应商听到"同行成本2元"，通常会让步。

### Q3: 极端折扣标签是什么意思？
**A**: 竞对在搞引流促销（1-3折），肯定亏本或微利。这种商品别盲目跟进，跟了就亏。

### Q4: 每周都要跑一次吗？
**A**: 建议每周一跑一次，10分钟而已。及时发现竞对价格变化，避免失去竞争力。

### Q5: 报告太长，看不过来怎么办？
**A**: 只看3张表（Sheet 1/6/13），其他表按需查看。每周花30分钟足够。
