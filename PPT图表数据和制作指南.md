## 📊 第2页：为什么要做这个工具？

### 图表1：以前 vs 现在对比表

**图表类型**：对比表格（左右分栏）

**Excel数据**：
```
┌─────────────┬──────────────────┬────────────────────┐
│   对比项    │   以前（手工）   │  现在（工具）      │
├─────────────┼──────────────────┼────────────────────┤
│ 时间成本    │   2-3天          │  10分钟 ⚡         │

└─────────────┴──────────────────┴────────────────────┘
```

**PPT制作步骤**：
1. 插入表格（5行×3列）
2. 第1列灰色底色，第2列红色底色（代表以前），第3列绿色底色（代表现在）
3. 字体：微软雅黑，标题16号加粗，内容14号
4. 在"现在"列关键数字后加闪电⚡图标

## 🎨 PPT整体设计建议

### 配色方案
```
主色：深蓝色 (#1E3A8A) - 专业、可信
辅色1：绿色 (#10B981) - 增长、正面
辅色2：橙色 (#F59E0B) - 警告、重点
辅色3：红色 (#EF4444) - 问题、风险
背景：白色或浅灰 (#F9FAFB)
```

### 字体规范
```
标题：微软雅黑 24-32号 加粗
副标题：微软雅黑 18-20号 加粗
正文：微软雅黑 14-16号 常规
数据：Arial 18-48号 加粗（数字用Arial更清晰）
```

### 图标资源
```
推荐网站：
  • iconfont.cn（阿里巴巴图标库，免费）
  • flaticon.com（扁平化图标，免费/付费）
  • icons8.com（多风格图标，免费/付费）

常用图标：
  ⚡ 效率提升
  💰 收益/成本
  📈 增长/ROI
  ✅ 成功/正确
  ❌ 失败/错误
  ⚠️ 警告/注意
  🎯 目标/精准
  🔜 未来/计划
```

### 动画建议
```
第1页（封面）：
  标题：淡入（0.5秒）

第2-7页（内容页）：
  标题：飞入（从左，0.3秒）
  图表：淡入+放大（0.5秒，延迟0.2秒）
  文字：逐行出现（向上擦除，0.2秒/行）

切换效果：
  推进（向左，0.5秒）

注意：不要过度使用动画，简洁专业最重要
```

## 🖨️ 打印版建议

如果需要打印PPT（如分发给参会人员）：

1. **纸张设置**：A4横向
2. **每页幻灯片数**：2张（上下排列）
3. **是否打印备注**：建议打印演讲稿作为备注
4. **黑白打印优化**：
   - 使用高对比度（深蓝vs白，而非彩色渐变）
   - 避免浅色背景（灰色在黑白打印时不明显）
