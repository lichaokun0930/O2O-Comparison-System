## 【第4页】每个指标什么意思？怎么用？

**PPT内容**：
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  报告里的关键指标解读（不懂就问）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

指标1：价格差异率
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
什么意思？
  (我们价格 - 竞对价格) / 竞对价格 × 100%
  
怎么看？
  +12% → 我们贵12%，要调价❌
  -12% → 我们便宜12%，可以主推✅
  
怎么用？
  运营筛选：价格差异率 > 10%的商品
  → 贵的：调价或下架
  → 便宜的：打"低价爆品"标签

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

指标2：预测成本
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
什么意思？
  根据竞对的价格，倒推他们的进货成本
  
怎么算？（不用懂技术，看结果就行）
  系统用行业加价率（1.5-2倍）反推
  比如：竞对卖3元，推测成本2元（加价1.5倍）
  
怎么用？
  对比我们的进货价：
    竞对成本2元，我们进货2.5元 → 被坑了25%
    找供应商谈判的底气："同行成本2元，你卖我2.5元？"
  
注意：
  每个预测都有"置信度"（0.5-0.95分）
  0.8以上 → 可信度高，可以用
  0.5-0.7 → 参考就行，别全信

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

指标3：预测方法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
什么意思？
  成本是怎么算出来的（标签，看结果就行）
  
常见标签：
  "三级分类(售价加权30%)" → 靠谱，可信✅
  "一级分类(原价)" → 一般般，参考
  "[调整:极端折扣18%]" → 竞对在搞促销，成本估不准⚠️
  
怎么用？
  看到"极端折扣"标签 → 竞对在引流，别跟进亏本促销
  看到"售价加权"标签 → 成本估得比较准，可以用来谈判

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

指标4：置信度
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
什么意思？
  成本预测的可信程度（0-1分，越高越准）
  
怎么看？
  0.9分 → 非常准，可以直接用来谈判✅
  0.7分 → 比较准，可以参考
  0.5分 → 仅供参考，别全信⚠️
  
怎么用？
  筛选置信度≥0.8的商品 → 优先谈判
  筛选置信度<0.6的商品 → 人工复核

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

运营部记住：
  • 价格差异率 → 调价决策（贵的降，便宜的推）
  • 预测成本 → 供应链谈判（对比进货价）
  • 置信度 → 判断可信度（高的能用，低的参考）
  • 极端折扣标签 → 别盲目跟进促销
```

**演讲稿**（1.5分钟）：
> "报告里有几个关键指标，运营部要懂什么意思。
> 
> **价格差异率**：
> - +12%代表我们贵12%，要调价
> - -12%代表我们便宜12%，打'低价爆品'主推
> - 运营部筛选差异率>10%的商品，重点处理
> 
> **预测成本**：
> - 这是最值钱的指标，告诉你竞对的进货成本大概多少
> - 比如竞对卖3元，系统推测成本2元
> - 对比我们进货价2.5元，就知道被供应商坑了
> - 谈判时直接说：'同行成本2元，你卖我2.5元？'
> 
> **置信度**：
> - 0.9分非常准，可以直接拿去谈判
> - 0.5分只能参考，别全信
> - 运营筛选置信度≥0.8的商品优先谈判
> 
> **极端折扣标签**：
> - 如果看到'调整:极端折扣18%'，说明竞对在搞引流促销
> - 这种别跟进，跟了就亏本
> 
> 运营部记住4个指标：价格差异率决定调价，预测成本用来谈判，置信度判断可信度，极端折扣别盲目跟进。"

  上线1个月，已经用了3次，看效果

  运营部：
    ✅ 120款便宜商品 → 首页推"低价爆品"专区
    ❌ 85款贵的商品 → 30款调价，55款下架
    
结果：
  • 首页点击率提升25%（低价爆品吸引眼球）
  • 销售额提升18%（120款爆品销量翻倍）
  • 投诉率下降15%（下架了55款"贵货"）
  
赚了多少？
  月增收：18% × 平均月销200万 = 36万元
  年增收：36万 × 12 = 432万元 ✅

案例2：品类补充（第2周）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
发现了什么？
  报告显示：竞对有32款"电解质水"
            我们0款，完全缺失
