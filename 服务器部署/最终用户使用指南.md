# O2O 比价工具 - 最终用户使用指南

## 🎯 重要提示：用户需要安装什么？

### **答案：什么都不需要！** ✅

对方只需要：
1. ✅ **浏览器**（Chrome、Edge、Safari、Firefox 都行）
2. ✅ **网络连接**（能上网即可）
3. ✅ **您分享的链接**（例如：`http://abc123.ngrok.io`）

### ❌ 不需要的东西：
- ❌ 不需要安装 Python
- ❌ 不需要安装任何编辑工具
- ❌ 不需要安装依赖库
- ❌ 不需要懂任何技术
- ❌ 不需要配置环境

## 📱 使用步骤（给最终用户看）

### 第1步：打开浏览器
- 任何浏览器都可以：Chrome、Edge、Safari、360浏览器...
- 手机、电脑、平板都能用

### 第2步：输入链接
- 复制管理员给您的链接，例如：
  - `http://abc123.ngrok.io`（内网穿透）
  - `http://123.456.789.0:8555`（云服务器）
  - `http://yourdomain.com`（绑定域名）

### 第3步：上传文件
1. 点击 **"上传本店数据"** 按钮
2. 选择您的 Excel 文件（.xlsx 格式）
3. 点击 **"上传竞对数据"** 按钮
4. 选择竞对的 Excel 文件

### 第4步：选择模型（可选）
- 推荐选择：**模型 5: BGE-M3**（准确率最高）
- 如果不确定，保持默认即可

### 第5步：点击"开始分析"
- 等待进度条走完（约 2-5 分钟）
- 过程中会显示：
  - ⏳ 准备环境...
  - ⏳ 加载模型...
  - ⏳ 读取数据...
  - ⏳ 智能匹配中...
  - ✅ 分析完成！

### 第6步：下载报告
- 点击 **"下载比价报告"** 按钮
- 文件自动保存到电脑（.xlsx 格式）
- 用 Excel 或 WPS 打开即可查看

## ❓ 常见问题

### Q1: 我打开链接提示"无法访问此网站"？
**A:** 检查以下几点：
- 链接是否完整复制？
- 管理员的应用是否在运行？
- 网络连接是否正常？

### Q2: 文件上传失败？
**A:** 确保：
- Excel 文件格式正确（.xlsx）
- 文件大小不超过 200MB
- 包含必需列：商品名称、原价、售价

### Q3: 分析进度卡住不动？
**A:** 
- 等待 2-5 分钟（大文件需要更长时间）
- 刷新页面重新上传
- 联系管理员检查服务器

### Q4: 下载的报告打不开？
**A:**
- 用 Excel 2016+ 或 WPS 打开
- 确保下载完整（文件大小 > 0）
- 重新下载一次

### Q5: 多人同时使用会冲突吗？
**A:** 
- 不会！每个人的分析是独立的
- 支持最多 10 人同时使用
- 超过请联系管理员升级服务器

## 📊 报告解读（输出文件说明）

下载的 Excel 文件包含 **9个工作表**：

| 工作表名称 | 说明 |
|-----------|------|
| 1️⃣ 条码精确匹配 | 完全相同的商品（通过条码识别） |
| 2️⃣ 智能模糊匹配 | AI 识别的相似商品（不同条码但属性相似） |
| 3️⃣ 差异品对比 | 同类但有差异的商品（如不同规格） |
| 4️⃣ 本店独有商品_全部 | 竞对没有的商品（原始数据） |
| 5️⃣ 竞对独有商品_全部 | 本店没有的商品（原始数据） |
| 6️⃣ 本店独有商品_去重 | 按商品名聚合后的本店独有品 |
| 7️⃣ 竞对独有商品_去重 | 按商品名聚合后的竞对独有品 |
| 8️⃣ 品类缺口分析 | 建议补充的商品品类 |
| 9️⃣ 折扣优势对比 | 价格优势分析 |

**查看技巧：**
- 先看 **条码精确匹配** 和 **智能模糊匹配**
- 重点关注 **价格差异** 列（标红的是竞对更便宜）
- **品类缺口** 显示战略补品建议
- **折扣优势** 帮助制定促销策略
