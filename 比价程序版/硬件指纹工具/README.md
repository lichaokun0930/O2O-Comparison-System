# 硬件指纹工具 - 独立版本

## 📌 工具说明

本工具用于生成当前计算机的唯一硬件指纹，用于O2O比价工具的授权验证。

**重要**: 此工具已独立打包，不会被主程序打包覆盖。

---

## 📦 打包方法

### 方式1：使用已有的打包脚本（推荐）

```powershell
# 在主目录执行
.\打包指纹工具.ps1
```

打包后的文件位置：
- `dist/硬件指纹工具/硬件指纹工具.exe`

### 方式2：手动打包

```powershell
# 进入工具目录
cd 硬件指纹工具

# 使用PyInstaller打包
pyinstaller --onefile --name "硬件指纹工具" --icon=icon.ico generate_fingerprint.py

# 打包完成后，exe文件在 dist/ 目录
```

---

## 🚀 使用方法

### 对于普通用户

1. 双击运行 `硬件指纹工具.exe`
2. 程序会显示您的硬件指纹
3. 将生成的 `my_fingerprint.txt` 文件发送给管理员
4. 管理员会根据指纹生成授权密钥

### 对于开发者

```powershell
# 直接运行Python脚本
python generate_fingerprint.py
```

## 🔧 维护说明

### 如何更新工具

1. 修改 `generate_fingerprint.py` 源代码
2. 重新打包：`.\打包指纹工具.ps1`
3. 新的exe会覆盖旧版本

### 如何分发给用户

**重要**：分发时只需要 `硬件指纹工具.exe`，无需其他文件。

1. 从 `dist/硬件指纹工具/` 复制exe文件
2. 发送给用户
3. 用户双击运行即可

## ⚠️ 注意事项

1. **独立存在**: 此工具目录不参与主程序打包，不会被覆盖
2. **版本管理**: 建议给每个版本的exe加上日期标记（如：硬件指纹工具_20251102.exe）
3. **保留历史版本**: 如需回滚，可以保留旧版本的exe
4. **打包环境**: 确保打包时使用的Python环境与主程序一致

## 🐛 已知问题

### 问题1：打包后exe文件消失
**原因**: 主程序打包时覆盖了dist目录  
**解决**: 已将工具独立到此目录，不再受影响

### 问题2：Windows Defender误报
**原因**: PyInstaller打包的exe可能被误报为病毒  
**解决**: 
- 添加到Windows Defender排除列表
- 或使用数字签名（需购买证书）
